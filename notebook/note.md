# note

## 常用命令
- vim `<C-z>`把前台任务放在后台 --clean 不加载配置文件
- jobs 查看后台任务 - fg %n 将后台任务移到前台 kill
- <C-x><C-e> 将命令行命令放在vim中编辑
- jq 格式化json
- procs 进程展示
- btm 类似htop
- etree 树形显示
- dust 显示磁盘占用
- duf 显示磁盘空间
- fastfetch 优雅的打印系统信息
- extract zsh解压插件
- tokei 分析统计代码工具
- tldr 平易近人的man手册
- type 获取命令来源 -a 查看所有定义
- zsh 插件sudo 使用两次esc来增加sudo

zsh
- <C-u> 清空当前行
- <C-a> 移动行首
- <C-e> 移动行末尾
- <C-f> 向前移动
- <C-b> 向后移动
- <C-r> 搜索历史
- <C-g> 逃离搜索模式
