# tips
change work dir: `cd %:h`

# vim易忘技巧
- `<C-z>`把前台任务放在后台 jobs查看后台任务 - fg %n 将后台任务移到前台 kill
- 搜索时可以用 `CTRL-R CTRL-W` 插入光标下的单词，命令模式也能这么用
- `CTRL-R "` 将复制的内容粘贴到命令模式
